# Dice Game

This is a simple game of chance where upon the roll of dice, the player with the greatest number wins. 


### Built with

- HTML 5
- CSS
- Javascript


### What I learned

- Making the website functional using javascript
- Random function in javascript
- The Document Object Model - querySelector, setAttribute, onClick
